# create-alchemy-bootstrap-ts 

## Instructions

1) Click the friendly "Use this Template" button. Create a new repo name and click "Create repository from this template".

1) Click the friendly, green "Code" button. Click the clipboard icon to get the URL.

1) In your terminal, `git clone <the copied repo URL>`

1) In your terminal, `cd <your repo name`

1) In your terminal, `npm i`

1) In your terminal, `code .` to open the text editor. 

1) In your terminal, `npm start`. A tab should open in your browser. Change files in `src/index.html` and `src/index.ts` to live update this code.
